# Debounce

The v-debounce directive allows you to defer an v-model update on an input field by an amount of time.

# API

options
api.name
value
api.default
500
Type
number

With native inputs: number of milliseconds to wait before emitting event.

api.name
value
api.default
undefined
Type
function

With custom inputs: callback of the debounced event.

arg:delay

Used with custom components. Number of milliseconds to wait before emitting event.
